# Security Policy

## Supported Versions

This is a static client-side web application for "Asser's Competition" quiz. As it's a simple project without formal versioning, all current deployments are supported. Security updates will be applied as needed.

| Component | Supported          |
| ---------- | ------------------ |
| HTML/CSS/JS | :white_check_mark: |
| Browser Compatibility | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

### How to Report
- **Email**: Send details to [your-email@example.com] (replace with actual contact).
- **Include**: Description of the vulnerability, steps to reproduce, potential impact, and any suggested fixes.
- **Response Time**: Expect an acknowledgment within 7 days. We'll investigate and respond with updates.

### What to Expect
- We'll confirm receipt and assess the issue.
- If accepted, we'll work on a fix and notify you when resolved.
- If declined, we'll explain why.
- Please do not publicly disclose until we've addressed it.

We appreciate your help in keeping the project secure!
